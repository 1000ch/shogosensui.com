Shogo Sensui

Web アプリケーション開発を専門とするソフトウェアエンジニア。HTML・CSS・JavaScript・ブラウザ API などの Web 標準技術に関する動向を探りながら、新しいブラウザ技術で Web アプリケーションの価値をどのように高められるかを模索している。

企業でソフトウェアエンジニアとして働く傍ら、技術顧問として複数企業のエンジニアリングに関わっており、高品質で維持しやすい Web アプリケーションを作るための活動を続けている。ブラウザの動作原理について詳しく、品質の中でも特にパフォーマンスやアクセシビリティといった非機能要件に長らく取り組んでいる。

カンファレンスや勉強会への登壇依頼・Web 開発全般に関する技術的なアドバイスや支援の依頼は、メールにてご連絡ください。
