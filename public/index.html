Shogo Sensui

Web アプリケーション開発を専門とするソフトウェアエンジニア。企業で働く傍ら、技術顧問として複数企業のエンジニアリングに関わり、高品質で維持しやすい Web アプリケーションを作るための活動を続けている。

HTML・CSS・JavaScript・ブラウザ API などの Web 標準技術の動向を追いつつ、新しいブラウザ技術で Web アプリケーションの価値をどのように高められるかを模索している。特にブラウザの動作原理について詳しく、品質の中でもパフォーマンスやアクセシビリティといった非機能要件に長らく取り組んでいる。
